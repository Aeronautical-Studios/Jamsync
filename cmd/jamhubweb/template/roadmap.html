Roadmap

Currently Jam and JamHub are in the early stages of development. You can use JamHub to version control your game, but it is not yet ready for production. Below is a list of features that are planned for the future.

Q3 2023

  • Merge messages
  • Change reviews
  • Commit reverts
  • Hard file locking

Q4 2023

  • Soft File locking
  • CI/CD integration
  • Unreal Engine Plugin Integration
  • Unity Plugin Integration
  • Standalone UI
  • Partial Cloning
