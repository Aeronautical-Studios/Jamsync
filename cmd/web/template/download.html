Installation and Usage

The Jamsync client is a single, small (~15MB) binary. To install, just download the client above and put it somewhere on your PATH.

See the help text for the `jam` binary for up-to-date descriptions of the commands.

For initial setup of a project, you'll need to run the following commands:

  • `jam login` - opens a browser session that should authenticate with your jamsync account.
  • `jam init` - prompts you to upload a new project or download an existing project.
  • `jam sync` - watches for file changes and updates files to the most recent change.

Contact zach@jamsync.dev if you need any assistance!
