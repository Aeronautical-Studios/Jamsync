Download

You can try out JamHub for yourself using the CLI binaries below. We make daily cloud backups but we're still under development so make sure to back up your files.

CLI Binaries

  • darwin/arm64
  • darwin/amd64
  • linux/arm64
  • linux/amd64
  • windows/arm64
  • windows/amd64

Installation and Usage

The JamHub client is a single, small (~15MB) binary. To install, just download the client above and put it somewhere on your PATH. If you need guidance on how to do this, follow this guide for Windows or this guide for Mac or Linux.

See the help text for the `jam` command for up-to-date descriptions of the commands.

Contact zach@jamhub.dev or reach out on Discord if you need any help!
